Especificaciones:
1-> Funcionamiento del Formulario de Inicio de Sesión
    • Se conecta directamente a la API (ver la sección de fuentes) y se siguen los pasos brindados por los desarrolladores.

2-> Funcionamiento del Formulario de Registro
    • Se crean los archivos JS y PHP.
    • Se codifican los algoritmos necesarios:
        • JS para la conexión con las etiquetas HTML y obtener los datos del formulario (revisar los archivos .js y .php).
        • Conexión desde JS a PHP para realizar la petición fetch (revisar los archivos .js y .php).
        • PHP para la conexión con la BBDD MYSQL (revisar los archivos .js y .php).
            • Tener algún servidor para realizar correctamente los métodos POST.
                • XAMPP es gratuito e intuitivo.
                    • Crear la carpeta del proyecto en la ruta:
                        • C:\xampp\htdocs\Gimnasio
                            • Encender el servidor Apache.
                            • Encender el servidor MySQL.
                                • Desde XAMPP u otro, por ejemplo Workbench.
                            • Realizar pruebas en el formulario.
                                • 100% funcional, pero admite cambios en tablas y campos de la BBDD.
                                    • Estos cambios generan más cambios en el resto de archivos (.html, .js, .php).
